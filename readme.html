Collection of personal dot files for ease of install across devices.

Config

Collection of dot files. All application specific configuration is intended to be under here (except in cases where applications are patched)

bin

Collection of special use executables. Basically they are many small wrappers of convince software that aren’t big enough to be their own projects Some highlights

  • attach – wrapper around abacdo
  • brownnoise/pinknoise – noise generator
  • delay – sleep some specified time then run specified command
  • highlight – highlight selected text
  • kiss-setup.sh – sets up system wide kiss repros
  • link_dot_files.sh – links the files in Config to ~/.config/
  • nmenu – an interactive menu selection based on config files; intended as a primary UI for mobile devices with few buttons
  • screenshot – take a screenshot/video
  • status-bar – wrapper around dzen for the 2 status bars used
  • tz_select – manually select timezone
  • update-timezone – update timezone based on region
  • vol – update volume and notify user
  • weather.sh – get weather info and cache it
  • ytmenu – search for youtube videos and plays the selected one
